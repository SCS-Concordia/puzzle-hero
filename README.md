## Quick Start

### Client
- `cd client`
- `npm install`
- `npm start`

### Server

- `python3 -m pip install -r server/requirements.txt`
- `SLACK_TOKEN=%enter_token_here% SPARKPOST_TOKEN=%enter_token_here% python3 server/src/run.py`
